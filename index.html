Sincronização (GitHub Gist)

Sincronize seus dados entre celular, tablet e computador usando um Gist privado do GitHub. Use o botão abaixo para criar o token já com a permissão gist ativada, copie o código gerado (ghp_...) e cole aqui.

Criar token no GitHub (escopo gist)
  1. Toque no botão acima (faça login no GitHub se pedir).
  2. Role até o fim e clique em Generate token.
  3. Copie o código ghp_... e cole no campo abaixo.
  4. Clique em Enviar para criar seu Gist privado.

Proteção do código-fonte

Camadas que dificultam a cópia casual do conteúdo e do código. Importante: por ser um app que roda no navegador, isto desencoraja, mas não impede totalmente — um usuário técnico ainda pode acessar o código. As alterações são aplicadas na hora.

Bloquear menu de contexto
Desativa clique direito e toque longo
Impedir seleção e cópia
Textos da interface não podem ser selecionados (campos de digitação seguem normais)
Bloquear atalhos de inspeção
F12, Ctrl+Shift+I/J/C, Ctrl+U (ver código) e Ctrl+S (salvar página)
Impedir arrastar/salvar imagens
Bloqueia o arrasto da logomarca e da assinatura
Detectar ferramentas de desenvolvedor experimental
Exibe um aviso se o DevTools for aberto. Pode gerar alarmes falsos.
